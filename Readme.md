# SWWW (CLI) Background setter

This is an app that help you modifying your actual swww wallpaper, BUT, This app is CLI  
It is just pure text

## Usage

Just run the program and it will create a file in ~/.local/state/wallpapersconfig.txt, There will be all yours backgrounds  
The file must be like this:
```
Documents/Wallpapers
Documents/Wallpapers/Wallpaper1.png
```

### IMPORTANT

If you want the program to maintain the background even if you reboot swww, Just instead of running swww, Run the file "~/.local/state/initback.sh"
